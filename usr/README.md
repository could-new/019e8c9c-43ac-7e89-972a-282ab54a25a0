# InstaAI

InstaAI is an Instagram-like social media application featuring an integrated AI Agent that assists users with content creation, automated posting, and chat interactions. It includes premium features locked behind a subscription paywall.

## Features

*   **Home Feed:** View posts from friends and AI-generated content in a familiar, scrollable interface.
*   **AI Agent Assistant:** A dedicated chat interface to interact with your personal AI agent for scheduling, content ideas, and generation.
*   **AI Content Creation:** Tools to generate images and write captions seamlessly.
*   **Profile Management:** Manage your posts, followers, and AI-automated content settings.
*   **Premium Subscriptions:** Unlock advanced generative capabilities and automated posting via a Pro subscription (mocked card entry included).

## Tech Stack

*   **Framework:** Flutter (Cross-platform: iOS, Android, Web)
*   **Language:** Dart

## Getting Started

1.  Ensure you have [Flutter installed](https://docs.flutter.dev/get-started/install).
2.  Clone the repository and navigate to the project directory.
3.  Run `flutter pub get` to install dependencies.
4.  Run `flutter run` to launch the app on an emulator or connected device.

## Deployment

To generate an APK for the Google Play Store:
```bash
flutter build apk --release
```
The resulting APK will be located at `build/app/outputs/flutter-apk/app-release.apk`.
